# 유스케이스 시나리오

---

## 액터

| 액터       | 설명                  |
| ---------- | --------------------- |
| 사용자     | 웹 브라우저 이용자    |
| Aladin API | 외부 도서 검색 API    |
| PocketBase | 백엔드 DB 서버 (8090) |
| OpenAI API | 표지 이미지 생성      |

---

## UC-01: 도서 목록 및 대시보드 조회

**기본 경로**

1. 사용자가 메인 페이지에 접속한다.
2. PocketBase에서 전체 도서 대시보드 정보 및 1페이지 도서 목록(8개)을 조회한다.
3. 도서 카드 그리드(목록 뷰)와 대출 현황 파이 차트가 표시된다.
4. 페이지네이션을 통해 다음 페이지의 도서를 조회한다.

**대안 경로**

- **정렬 변경**: "최신순 / 오래된 순 / 제목순" 버튼 클릭 → 해당 기준으로 재정렬 및 1페이지 갱신
- **도서 0건**: 등록된 도서가 없을 경우 차트는 렌더링되지 않음

## UC-02: 사용자 인증 (회원가입/로그인/로그아웃)

**기본 경로**

1. 비로그인 상태에서 헤더의 '로그인' 또는 '회원가입' 버튼을 클릭한다.
2. 모달창에 정보를 입력하고 Submit한다.
3. PocketBase Auth API를 통해 인증 성공 후 브라우저 쿠키에 인증 상태가 저장된다.
4. 사용자 이름 및 마이페이지 버튼이 상단에 노출된다.

---

## UC-03: 도서 검색 및 등록 (+ Search / + Creator)

**기본 경로 (API 검색 등록)**

1. 사용자가 (로그인 후) "+ Search" 버튼을 클릭한다.
2. 도서 검색 모달에서 키워드를 입력 후 검색한다.
3. Aladin API에서 결과를 받아 목록에 표시한다.
4. "등록" 클릭 시 상세 지표를 조회한 후 PocketBase DB에 생성 요청을 보낸다.
5. 등록 완료 후 모달이 닫히며 도서 목록이 즉시 갱신(invalidate)된다.

**대안 경로 (수동 등록)**

1. 사용자가 (로그인 후) "+ Creator" 버튼을 클릭한다.
2. 수동 등록 폼에 도서 정보를 직접 기입한다.
3. "등록" 클릭 시 PocketBase DB에 도서 데이터를 저장한다.

## UC-04: 도서 상세 조회 및 상호작용 (대출/삭제/좋아요)

**기본 경로**

1. 목록에서 도서를 클릭하면 메인 뷰가 '상세 정보 뷰'로 전환된다.
2. 사용자가 '대출 가능/대출 중' 토글 버튼을 클릭하여 DB에 PATCH 요청을 보낸다.
3. 도서 카드의 '좋아요' 버튼을 클릭하면 `likes` 컬렉션에 등록되며 좋아요 수가 증가한다.
4. 작성자의 경우 '삭제' 버튼을 눌러 도서를 시스템에서 완전 삭제할 수 있다.
5. "목록으로 돌아가기" 버튼으로 메인 뷰로 복귀한다.

**대안 경로**

- **삭제 시**: Confirm 창에서 취소 가능. 확인 시 삭제되며 대시보드 차트와 목록이 함께 갱신됨.

---

## UC-05: AI 표지 이미지 생성

**기본 경로**

1. 사용자가 도서 상세 뷰에서 'AI 표지 생성' 탭을 활성화한다.
2. 프롬프트를 입력하고 '표지 이미지 생성하기'를 클릭한다.
3. 백엔드 Next.js API(/api/genthum)가 OpenAI gpt-image-2에 요청을 보낸다.
4. 생성된 이미지가 로컬 `public/covers/`에 저장되고, DB `thumbnail` 필드에 상대 경로가 PATCH된다.
5. 성공 알림과 함께 상세 뷰 썸네일 이미지가 최신화된다.

---

## UC-06: 실시간 인기 도서 랭킹 조회

**기본 경로**

1. 메인 페이지 로드 시 좌측 랭킹 사이드바 컴포넌트가 활성화된다.
2. 내부적으로 `getFullList`를 통해 전체 도서를 조회하여 좋아요 수 기준 내림차순 상위 10개를 필터링한다.
3. 1, 2, 3위는 트로피 아이콘으로 표시되며 랭킹 리스트가 렌더링된다.
4. 랭킹의 도서를 클릭하면 메인 화면이 해당 도서의 '상세 뷰'로 전환된다.

**대안 경로**

- **모바일 환경**: 좌측 사이드바가 숨겨지고 하단에 플로팅 '도서 랭킹' 버튼이 나타남. 클릭 시 모달창으로 랭킹 확인 가능.
