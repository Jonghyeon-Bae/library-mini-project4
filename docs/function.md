# 도서 관리 시스템 기능 명세서

## 도서 목록 조회 기능

- 시스템에 등록된 전체 도서 목록 화면 출력
- 드롭다운 필터를 제공하여 조건별 도서 조회 기능 (GET)
- 이름순, 최신순 정렬 옵션 제공
- 특정 도서 클릭 시 해당 도서의 '상세 정보 조회 페이지' 모달 호출
- 로그인한 사용자가 본인이 작성/등록한 도서들만 모아서 볼 수 있는 내 도서 목록 페이지 제공 (전체 목록과 동일한 기능)

---

## 신규 도서 등록 기능

- '작성' 버튼 클릭 시 도서 등록 폼 제공
- 제목, 작성자, 상세정보, 본문 필수 입력
- 썸네일이 비어있는 상태로는 도서 등록 불가
- 사용자가 직접 이미지를 업로드하거나 'AI 이미지 자동 생성' 기능을 반드시 사용해야 함
- 작성일, 수정일은 폼에서 노출 및 입력하지 않으며 등록 시 자동 기록됨

---

## 도서 상세 정보 렌더링 기능

- 도서의 전체 정보(제목, 작성자, 상세정보, 본문, thumbnail, 작성일, 수정일) 화면 노출
- 화면 좌측에 도서 thumbnail 이미지 배치
- 화면 우측에 상세 정보와 본문 텍스트 배치

---

## 도서 수정 및 삭제 기능

- 본인이 작성한 도서 상세 정보 화면에 수정 및 삭제 버튼 제공
- '수정' 버튼 클릭 시 화면 내 정보가 입력/수정 가능한 폼 형태로 전환
- 수정 모드 진입 시 기존 '수정' 버튼은 '등록(또는 저장)' 디자인으로 변경
- 변경된 데이터를 PocketBase로 PATCH 전송
- 새 이미지를 직접 업로드하거나 AI로 재성성하여 기존 썸네일 교체 가능
- '삭제' 버튼 클릭 시 해당 도서 데이터 삭제 DELETE 요청 전송

---

## AI thumbnail 이미지 생성 기능

- 도서 상세/수정/등록 페이지 내 '썸네일 이미지 생성' 버튼 제공
- 현재 입력/등록된 도서의 제목, 상세정보, 본문/리뷰 텍스트를 분석하여 핵심 키워드 추출
- 추출된 키워드를 기반으로 이미지 생성 모델에 1024x1536 규격 이미지 생성 요청
- 생성되는 이미지는 도서 제목과 내용을 직관적으로 반영해야 함
- 생성된 이미지를 미리보기 화면으로 제공하여 사용자 승인 유도
- 사용자 승인 시 해당 이미지가 썸네일 폼 데이터에 적용됨
- 기존 썸네일이 없으면 create 방식으로 파일 첨부, 존재하면 기존 파일을 대체하는 update 방식으로 적용
