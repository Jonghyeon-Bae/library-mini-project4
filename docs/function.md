# 도서 관리 시스템 기능 명세서

## 회원가입 및 로그인 기능

- 이메일 기반의 사용자 회원가입 및 로그인 기능 제공 (PocketBase Auth)
- 로그인한 사용자만 도서 등록, 대출 토글, 수정/삭제, 좋아요 등 쓰기 작업 수행 가능
- 로그인 후 본인 프로필 정보 및 등록한 도서를 확인할 수 있는 '마이페이지' 제공

---

## 도서 목록 조회 기능

- 시스템에 등록된 전체 도서 목록 화면 출력 (8개 단위 페이지네이션 지원)
- 대시보드 차트(파이 차트)를 통해 전체 도서의 대출 가능/대출 중 비율 시각화
- 드롭다운 필터를 제공하여 조건별 도서 조회 기능 (최신순, 오래된 순, 제목순)
- 도서 카드 클릭 시 화면 전환 없이 '상세 정보 뷰' 컴포넌트로 전환 (BookDetailView)
- 로그인한 사용자가 본인이 작성/등록한 도서들만 모아서 볼 수 있는 마이페이지 내 도서 목록 제공

---

## 인기 도서 랭킹 사이드바

- 좌측 플로팅 영역에 '인기 도서 TOP 10' 사이드바 제공
- 전체 도서 중 좋아요(like_count) 수가 가장 많은 10개 도서를 내림차순 정렬하여 렌더링
- 페이지 이동이나 페이지네이션 필터와 무관하게 전체 도서 데이터를 기준으로 실시간 랭킹 산정
- 모바일/태블릿 뷰에서는 플로팅 토글 버튼 및 모달 형태로 랭킹 제공

---

## 신규 도서 등록 기능

- '+ Search' 버튼 클릭 시 도서 검색 및 등록 모달 제공
- 검색어 입력 후 알라딘 API를 통해 실제 도서 데이터 검색
- 검색된 도서 목록 중 선택하여 '등록' 버튼 클릭 시 PocketBase DB에 저장

- '+ Creator' 버튼 클릭 시 도서 정보 수동 입력 모달 제공
- 도서 등록 시 초기 상태는 '대출 가능'으로 자동 설정

---

## 도서 상세 정보 뷰 (렌더링)

- 메인 페이지에서 도서 선택 시 목록 대신 상세 정보 뷰 렌더링
- 도서의 전체 정보(제목, 저자, 출판사, 본문, thumbnail) 화면 노출
- 화면 우측에 대출 상태 변경, 도서 삭제(작성자 전용), 목록으로 돌아가기 버튼 제공
- 도서 상세 정보 페이지 내에서 AI 표지 이미지 재생성 기능 제공

---

## 도서 좋아요 및 대출 상태 토글

- 도서 카드 하단 및 상세 뷰에 '좋아요' 하트 아이콘 제공
- 클릭 시 사용자-도서 매핑 데이터를 PocketBase `likes` 컬렉션에 추가/제거 (Optimistic UI 적용)
- 대출 가능/대출 중 버튼을 통해 현재 대여 상태 토글 (isAvailable 필드 업데이트)
- 대출 토글 또는 삭제 작업 시 랭킹과 대시보드 차트 데이터가 자동 갱신됨

---

## 도서 수정 및 삭제 기능

- 본인이 등록한 도서에 한해 상세 뷰에서 '삭제' 권한 제공
- '삭제' 버튼 클릭 시 Confirm 창 후 DELETE API 전송 및 도서 데이터 완전 삭제
- (수정 기능은 현재 썸네일 재작성 및 대출 상태 변경으로 대체됨)

---

## AI thumbnail 이미지 생성 기능

- 도서 상세 뷰에서 'AI 표지 다시 그리기' 기능 제공
- OpenAI gpt-image-2 API를 호출하여 도서 제목과 저자를 기반으로 이미지 생성
- 생성된 이미지는 서버를 거쳐 프로젝트 내 `public/covers/` 디렉토리에 파일로 직접 저장됨
- PocketBase DB에는 해당 파일의 로컬 상대 경로(`/covers/파일명.png`)를 업데이트
- 썸네일 생성 완료 시 도서 목록 및 상세 뷰에 즉시 최신 이미지가 렌더링됨
