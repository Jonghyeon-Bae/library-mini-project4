# 도서 관리 시스템 요구사항 분석서

## 시스템 개요

- 사용자가 도서를 등록하고 관리할 수 있는 웹 기반 플랫폼 제공
- AI 이미지 생성 모델을 활용하여 사용자가 손쉽게 도서의 썸네일 표지를 제작하도록 지원

---

## 사용자 목표

- 자신의 도서 목록을 체계적으로 등록, 조회, 수정, 삭제 가능
- 썸네일이 없는 사용자를 위해 도서의 제목과 본문 내용을 기반으로 AI가 자동 표지 이미지 생성
- 정렬 및 필터링(전체, 본인 도서)을 통해 원하는 도서를 빠르게 탐색

---

## 데이터베이스 환경

- 데이터베이스로 PocketBase 사용
- 데이터 조회 및 조작은 PocketBase의 RESTful API (CRUD) 통신 규칙 준수
- 초기 데이터는 kakao api를 통해 구축

---

## 데이터 구조 정책

- 모든 도서 데이터는 `books` 컬렉션 내에서 중앙 집중적으로 관리
- 이미지 파일 관리를 위해 기존 `cover` 필드 대신 PocketBase의 File 타입 필드인 `thumbnail` 사용
- `작성일`과 `수정일` 데이터는 생성/수정 시 PocketBase 시스템에서 자동 업데이트 및 관리

---

## 외부 API 연동

- 썸네일 자동 생성을 위해 외부 AI 이미지 생성 모델(GPT Image-2) API 연동
- 생성되는 이미지는 1024x1536 해상도 준수
